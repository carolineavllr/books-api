## Tecnologias
- React
- Styled Components

## Funcionamento
- Preencher o campo de busca com o nome do livro que desejar pesquisar
- Livros serão listados à baixo
- Ao clicar na imagem do livro, abrirá um popup com mais informações

## Getting Started

First, run the development server:

```bash
npm start
```
